Guide:


This calculator can be used to calculate the perimeter or length of any natural object. The length of any object depends upon the scale at which you measure it. For example, the perimeter of great britain is 2500km when measured with a 250km ruler, yet when using a 50km ruler, its perimeter is 4225km.

This calculator calculates the fractal dimension for all entered data, and uses this to calculate a fractal dimension for any entered scale. Because it uses multiple fractal dimensions, you can model any object (fractal and non fractal), and find its perimeter at any scale.

Step 1: Measure your object using different unit lengths (ex. 1km and 2km lengths)

Step 2: Record the numbe of times each unit length fits into your objects (ex your pond, river, coastline, etc).

Step 3: Input values from step 1 and 2, choose a scale, and press calculate
